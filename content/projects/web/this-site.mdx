Designed based on my Arch/i3 config - available with scripted install [here](/projects/systems/dotfiles).

## Stack

- Vite 6, React 19, TypeScript
- Tailwind CSS v4
- react-router-dom v7

## Content pipeline

- MDX via `@mdx-js/rollup`
- `remark-frontmatter` + `remark-mdx-frontmatter` for post metadata
- `remark-math` + `rehype-katex` for LaTeX
- `@shikijs/rehype` (`vitesse-black` theme) for syntax highlighting
- Posts auto-discovered via `import.meta.glob` — folder tree `content/<kind>/<category>/<slug>.mdx` is the URL tree

## Layout

- Single-tree site map (press `s`) and command menu (`h`)
- Colour cycling via `c` — six accent hues, persisted in localStorage
- Responsive: top nav on desktop, fixed bottom bar on mobile

## Extended functionality
- Try `h`

## Deploy

- Netlify, auto-build on push to `main`
- `_redirects` for SPA routing
